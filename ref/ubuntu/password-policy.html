Password Policy

Location

  1. Edit /etc/pam.d/common-password
  2. Edit /etc/login.defs

Settings

For /etc/pam.d/common-password
  1. Find the following line in the file:
    • password    requisite    pam_cracklib.so retry=3 minlen=8 difok=3
  2. To require password complexity, add to the end of the line:
    • ucredit=-1 lcredit=-1 dcredit=-1 ocredit=-1
  3. The line should now look like the following
    • password    requisite    pam_cracklib.so retry=3 minlen=8 difok=3 ucredit=-1 lcredit=-1 dcredit=-1 ocredit=-1
  4. Find the following line in the file:
    • password    [success=1 default=ignore]    pam_unix.so obscure use_authtok try_first_pass sha512
  5. To remember last 5 passwords, add to the end of the line:
    • remember=5
  6. To require at least 8 characters, add to the end of the line:
    • minlen=8
  7. The line should now look like the following
    • password    [success=1 default=ignore]    pam_unix.so obscure use_authtok try_first_pass sha512 remember=5 minlen=8
  8. Save the file
For /etc/login.defs
  1. Find PASS_MAX_DAYS, set to 90
  2. Find PASS_MIN_DAYS, set to 10
  3. Find PASS_WARN_AGE, set to 7
