Services

Location

  1. Go to Administrative Tools
  2. Find and open Services

Settings

Make sure all CyberPatriot services have Started as Status and Automatic as Startup Type.

Unless required by readme, stop each of the following services and change Startup Type as Disabled
  • DHCP Server
  • DNS Server
  • FTP Server
  • HTTP Server
  • Internet Information Services (IIS)
  • Telnet Server
