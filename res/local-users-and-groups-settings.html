Users

For each user, right click on user, select Properties, and check the following:

Under General tab
  • Disable all users not required by readme
  • Set passwords to expire for all users
Under Member Of tab
  • Add user to necessary groups, e.g. Administrators
  • Remove user from unnecessary groups

Groups

Check the following groups.
  • Administrators
    • only the users suppose to be here
  • Users
    • only the users suppose to be on the system
  • Backup Operators, Guests, Power Users
    • no users
  • Other groups
    • check as needed
