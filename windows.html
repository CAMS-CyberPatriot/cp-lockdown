Account Lockout Policy

Windows 7, Windows Server 2003, Windows Server 2008 R2

  1. Open Local Security Policy
  2. Open Account Policies
  3. Open Account Lockout Policy
  4. Change the following settings
    • Account lockout duration - 30 minutes
    • Account lockout threshold - 5
    • Reset account lockout counter after - 30 minutes

Administrative Tools

Windows Server 2003

  1. Press Start or Windows button
  2. Find and move mouse over Administrative Tools

Windows 7, Windows Server 2008 R2

  1. Open Control Panel
  2. Open System and Security
  3. Open Administrative Tools

Control Panel

Windows Server 2003

  1. Press Start or Windows button
  2. Find and move mouse over Control Panel

Windows 7, Windows Server 2008 R2

  1. Press Start or Windows button
  2. Find and click on Control Panel

Local Security Policy

Windows 7, Windows Server 2003, Windows Server 2008 R2

  1. Go to Administrative Tools
  2. Open Local Security Policy

Password Policy

Windows 7, Windows Server 2003, Windows Server 2008 R2

  1. Open Local Security Policy
  2. Open Account Policies
  3. Open Password Policy
  4. Change the following settings
    • Enforce password history - 5 passwords remembered
    • Maximum password age - 30 days
    • Minimum password age - 5 days
    • Minimum password length - 8 characters
    • Password must meet complexity requirements - Enabled
    • Store passwords using reversible encryption - Disabled
