Find Unauthorized Files

  1. Open Command Prompt
  2. dir C:\Users /s /b /a | find /i ".mp3"
    • this will search for .mp3 files in C:\Users
  3. If there are any results, inspect the files and remove as necessary
  4. Repeat from step 2 for the following file extensions
    • (music) - .mp3, .wav, .wma, .aac
    • (movie/video) - .mp4, .mov, .avi, .avi
    • (pictures) - .gif, .jpg, .jpeg, ,.png, .bmp
    • (executable files) - .exe, .msi, .bat, .sh

Windows Firewall

Location

Option 1
  1. Go to Control Panel
  2. Go to Network Connections
  3. Open Local Area Connection
  4. Find and press Properties
  5. Go to the Advanced tab
  6. If applicable, put a checkmark for Protect my comptuer and network by limiting or preventing access to this computer from the Internet
  7. Press Settings
Option 2
  1. Open Control Panel
  2. Find and click on Windows Firewall

Settings

  • Under the General tab
    • [x] On
    • [ ] Don't allow exceptions
  • Under the Exceptions tab
    • Only required programs or services should have checkmarks
    • Uncheck all other programs and features
  • Under the Advanced tab
    • Put a checkmark on all Network connections
