Checklist: Windows
__:___ read readme/instructions
Users and Groups
__:___ verify local user accounts, update passwords
__:___ verify local user groups
__:___ turn off automatic login
Installation Cleanup
__:___ check Add/Remove Programs
__:___ check Windows Features
__:___ check start up programs
__:___ check Services
__:___ check Task Scheduler
__:___ run disk cleanup
Networking
__:___ turn off Internet Connection Sharing
__:___ disable Remote Assistance and Remote Desktop
__:___ turn on Windows Firewall, check exceptions
__:___ check shared folders
__:___ monitor network connections (e.g. CrowdInspect, netstat, tcpview)
Policies
__:___ password policy
__:___ account lockout policy
__:___ audit policy
__:___ user rights assignment
__:___ security options
__:___ advanced audit policy configuration
__:___ local group policy editor
Scans
__:___ run Microsoft Baseline Security Analyzer without security updates check
__:___ run quick/full scan with anti-virus tool (e.g. Windows Defender, etc.)
__:___ run anti-malware tools (e.g. Norton Power Eraser, MalwareBytes)
Filesystem
__:___ show hidden files and folders
__:___ find and remove unauthorized files (e.g. pictures, music, programs)
__:___ check users folder (including Desktop)
__:___ check C:\ drive and Program Files folders
__:___ reset C:\windows\system32\drivers\etc\hosts to default
Updates
__:___ install latest service pack
__:___ turn on automatic updates
__:___ install only critical or important Windows updates
Configuration
__:___ reset web browsers (Internet Explorer, Mozilla Firefox, Google Chrome)
__:___ configure required roles for Windows Server
__:___ configure required services and programs
Logs
__:___ Event Viewer
